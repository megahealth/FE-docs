## 呼吸波数据接口

**简要描述：** 

- 呼吸波数据接口

**请求URL：** 

- ` http://shcparse.leanapp.cn/webApi/breathWave `

**请求方式：**

- GET

**参数：** 

| 参数名 | 必选 | 类型   | 说明   |
| :----- | :--- | :----- | ------ |
| id     | 是   | string | 报告ID |

 **返回示例**

```json
{
    "wave": ""
}

```

 **返回参数说明** 

| 参数名 | 类型  | 说明                                                         |
| :----- | :---- | ------------------------------------------------------------ |
| wave   | array | 呼吸波二维数组，数组元素为每分钟的呼吸波数据，1200个左右数据 |
